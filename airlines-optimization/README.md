# Airlines Visualizer (static)

- `index.html` をブラウザで開いて使います（ローカルで完結します）。
- **Download**: 入力生成器に基づいてケースを生成して `.txt` をダウンロードします（複数ケースは空行で区切って連結します）。同時に 1 件目を Input に反映します。
- **Visualize!**: Input/Output を解析して、スコア・スケジュール図・シェア棒グラフ・ランダム比較表を表示します。
- **Save as PNG**: スケジュール図を PNG で保存します。

## 実装している表示
1. スコア表示（不正な出力のときは score=0 とエラー内容を表示）
2. ユーザー出力（サークル航空）のフライトスケジュール図示
3. シェア量（v_sq / v_ci を 1e10 で割った値）棒グラフ
4. 人口比例でランダムに選んだ (a,b,t) について、スクエア/サークルそれぞれの「間に合うために必要な出発時刻」を表示

## 注意
- Random Moves はスコア計算と同じく「都市間距離が 0.25R 以上」の組からサンプルしています。
- 出力が空 (全機 c_i=0) の場合も仕様上は正当な出力なので、share=0 のまま表示します。
